=== WPMailing ===
Tags: users, groups, email
Tested up to: 2.2.1
Stable tag: /trunk/

O WPMailing permite criar grupos para organizar os usuários e enviar emails para eles por grupos ou por nível de usuário, capacidades ou login. É possível também enviar emails para não usuários, diretamente pelos endereços de email. Além disso, pode-se usar informações do usuários e dados de uma publicação escolhida (post ou página) para personalizar a mensagem e até incluir arquivos como anexos, se for o caso.
O WPMailing tem suporte para os plugins Xinha4WP e Dean's FCKEditor, para que você possa escrever e-mails no seu editor predileto.
  
== Instalação ==  

1. Descompacte o arquivo e copie a pasta wpmailig inteira para o diretório /wp-content/plugins/
1. Vá até a administração do WorPress, na aba Plugins, e ative o WPMailing  
1. Pronto!
  
== Características ==  
  
Uma vez ativado o plugin, seus painéis administrativos irão oferecer:

* A possibilidade de criar grupos e aplicá-los aos seus usuários, através de funcionalidades 
  adicionadas às páginas da seção Usuários (user.php, profile.php and user-edit.php). 
** Essas modificações são executadas com Javascript e **não modificam os arquivos originais do WP**.
* Interface para editar os nomes e descrições dos grupos (Manage > Groups)
* Interface para escrever e enviar emails aos usuários registrados no blog, 
  com algumas funcionalidades interessantes (Write > Write e-mail):
** O campo "Para" pode receber múltiplos valores, entre endereços de email e adição em 
   massa por grupo, capacidade ou nível dos usuários, ou individualmente, pelo login.
** O sistema permite anexar múltiplos arquivos à mensagem
** O e-mail pode conter valores de um determinado post ou página
** O e-mail pode conter valores personalizados do usuário que receberá a mensagem
* Interface para modificar algumas configurações do WPMailing (Opções > E-mail)
* Um relatório completo sobre o envio depois de enviar qualquer email.
